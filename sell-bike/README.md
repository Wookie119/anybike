# Motorcycle Selling Form Module

This is a **removable feature module** for anybike.co.uk. All files are contained in this directory.

## Quick Removal
If you don't like this feature:
1. Delete this entire `/sell-bike` folder
2. Delete the branch: `git branch -D feature/bike-selling-form`
3. Done! ✓

## Files in this Module
- `index.html` - Main form page
- `styles.css` - Form styling
- `script.js` - Form logic & image upload
- `database-schema.sql` - Database setup (optional)
- `api-backend.js` - Node.js backend example

## Features
✅ Step-by-step form wizard
✅ Image upload (up to 10 photos)
✅ Instant valuation calculation
✅ Responsive mobile design
✅ Data encryption ready
✅ No external dependencies

## Quick Start
1. Copy all files to your web server
2. Open `index.html` in browser to test
3. Configure backend API endpoint in `script.js` (line 15)
4. Set up database (optional)

## Integration
- Link from main site: `<a href="sell-bike/index.html">Sell Your Bike</a>`
- Or integrate form into existing page
